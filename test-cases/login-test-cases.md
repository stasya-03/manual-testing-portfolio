# Login Test Cases

---

### TC-001 Успешный вход

**Предусловие:** пользователь зарегистрирован  

**Шаги:**
1. Открыть страницу логина  
2. Ввести корректный email  
3. Ввести пароль  
4. Нажать "Войти"  

**Ожидаемый результат:**  
Пользователь успешно авторизован  

---

### TC-002 Неверный пароль

**Шаги:**
1. Ввести корректный email  
2. Ввести неверный пароль  
3. Нажать "Войти"  

**Ожидаемый результат:**  
Появляется сообщение об ошибке

### TC-003 Пустые поля
