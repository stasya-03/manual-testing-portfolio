# E-commerce Web Application Testing

Проект ручного тестирования веб-приложения интернет-магазина.

---

## 🌐 Объект тестирования

E-commerce веб-приложение

👉 https://henderson.ru

---

## 🎯 Цель

Проверка основного функционала интернет-магазина и выявление дефектов.

---

## 📂 Что было сделано

- анализ пользовательских сценариев  
- составление тест-кейсов  
- проведение тестирования  
- оформление баг-репортов  
- подготовка итогового отчета  

---

## 🧪 Область тестирования

- регистрация и авторизация  
- валидация форм  
- навигация  
- пользовательские сценарии  
- базовые функции интернет-магазина  

---

## 📁 Структура

- test-cases — тест-кейсы  
- bug-reports — баг-репорты  
- test-plan — план тестирования  
- checklist — чек-листы  
- summary-report — отчет  

---

## 📊 Результат

- покрыты основные пользовательские сценарии  
- выявлены дефекты
