# Manual Testing Portfolio

Репозиторий с примером ручного тестирования веб-приложения интернет-магазина (E-commerce).

---

## 🔥 Основной проект

В рамках курсовой работы был выполнен полный цикл тестирования веб-приложения:
от анализа требований до подготовки тестовой документации и отчетности.

---

## 🌐 Объект тестирования

Веб-приложение интернет-магазина (E-commerce)

Тестирование проводилось на реальном сайте:  
👉 https://henderson.ru

### Фокус тестирования:
- регистрация и авторизация пользователя  
- работа форм и валидации  
- навигация по сайту  
- пользовательские сценарии  
- базовые элементы оформления заказа
